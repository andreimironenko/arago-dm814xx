Welcome - Windows Users

Thank you for choosing the DM37x Evaluation Module (EVM). 

IMPORTANT NOTE - This Linux Software Development Kit (SDK) was developed for a Linux host machine.  If you are viewing this page in a Windows machine then there are 2 Linux partitions on this SD card that are not visible to Windows.  You will need to insert the USB SD Card Reader into a Linux machine.  If you do not have a Linux host machine you can Create a Linux host machine on your Windows PC.
