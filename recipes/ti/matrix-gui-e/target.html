Welcome

Thank you for choosing the C6A811x Evaluation Module (EVM).  This is the EVM version of the same setup.htm file you will find in the START_HERE partition once you power down the EVM, and insert the SD card into a Linux host machine. 

Starting your Development - In 2 Easy Steps!

1.  Software Developer's Guide at section How to create an SD card - Once you prepare the SD card, boot your EVM using this SD card, On bootup you will notice the matrix launcher demos showing up on the display, you can browse through each of the icons and explore various example applications and demos we have provided on the C6A811x EVM.

 

2.  Once you have enjoyed demo please refer to the printout of the C6A811x EVM Software Developer's Guide.  This document is designed to walk you through the easy process of starting your development on the C6A811x EVM.
